# Todo Backend (Node.js)

간단한 Node.js HTTP 서버로 구성된 Todo 백엔드 프로젝트입니다.

## 1. Node.js 설치 확인

PowerShell 또는 CMD에서 아래 명령으로 Node.js가 설치되어 있는지 확인하세요.

```bash
node -v
npm -v
```

버전이 나오지 않으면 Node.js 공식 사이트에서 LTS 버전을 설치하세요.

```text
https://nodejs.org
```

## 2. 프로젝트 실행 방법

1. 프로젝트 폴더로 이동

```bash
cd "c:\Users\TEHERAN\내 드라이브\03_Lab\01_Ai_autotest\todo-backend"
```

2. 서버 실행

```bash
npm start
```

또는

```bash
node index.js
```

3. 브라우저에서 아래 주소로 접속

```text
http://localhost:3000
```

정상적으로 실행되면 JSON 응답이 표시됩니다.
